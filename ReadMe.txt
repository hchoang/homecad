Guide to user Program:

-Open the Programe, first user have to enter name and budget of owner
	+name and budget must not be null
	+user can only enter number for bugget and the number must be lower than 1 billion
-Once the map show up, user can begin to design house by
	+double click on the empty panel or go to Edit - Add Room  to add room
	+double click on the room or go to Edit - Remove Room to Remove Room
	+go to Edit - Add Exit point to add exit point
	+go to Edit - Add Item to add item to room
	+go to Edit - Remove Exit Point to remove exit point
	+go to Edit - Remove Item to remove item
	+User can go to Edit - Reset to reset all setting to the time user beginning design the house
-To exit the programe, user can go File - Exit
